# Key insight from context

- `examples/proj/01-mercator.kt` has `@file:JvmName("Mercator")` â†’ correct class is `examples.proj.Mercator` NOT `examples.proj.MercatorKt`

The "To Run" comment block format is consistent:
```kotlin

 * ### To Run
 * ```
 * ./gradlew run -Popenrndr.application=examples.proj.MercatorKt

```

This needs to become `examples.proj.Mercator` (without `Kt` suffix).
