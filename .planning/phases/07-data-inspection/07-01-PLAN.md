**Implementation approach (from RESEARCH.md):**

1. **Early return for empty source:**
   ```kotlin
   if (isEmpty()) {
       println("GeoSource: Empty (no features)")
       return
   }
   ```

2. **Single-pass statistics collection** - iterate features ONCE, collecting:
   - Feature count (Long)
   - Combined bounds (use `Bounds.empty()` and `expandToInclude`)
   - Geometry type counts (Map<String, Int>) - use sealed class `when` to get type name
   - Property keys → inferred types (Map<String, String>)
   - Coordinate count for memory estimation

3. **Geometry type detection** - use exhaustive `when` on sealed class:
   ```kotlin
   val geomType = when (feature.geometry) {
       is Point -> "Point"
       is LineString -> "LineString"
       is Polygon -> "Polygon"
       is MultiPoint -> "MultiPoint"
       is MultiLineString -> "MultiLineString"
       is MultiPolygon -> "MultiPolygon"
   }
   ```

4. **Property type inference** - use Kotlin `when` with `is` checks:
   ```kotlin
   fun inferTypeName(value: Any?): String = when (value) {
       null -> "null"
       is String -> "String"
       is Int -> "Int"
       is Long -> "Long"
       is Double -> "Double"
       is Boolean -> "Boolean"
       is List<*> -> "List"
       is Map<*, *> -> "Map"
       else -> value::class.simpleName ?: "Any"
   }
   ```

5. **Coordinate counting** for memory estimate:
   - Point: 1
   - LineString: points.size
   - Polygon: exterior.size + interiors.sumOf { it.size }
   - Multi*: sum of contained geometries

6. **Memory estimation formula:**
   - ~24 bytes per coordinate (2 doubles + object overhead)
   - Format: "~N B", "~N KB", or "~N MB" based on size

7. **Output format** - box-drawing characters for clean visual:
   ```
   ┌──────────────────────────────────────────────────┐
   │               GeoSource Summary                  │
   ├──────────────────────────────────────────────────┤
   │ Features:    42                                  │
   │ CRS:         EPSG:4326                           │
   │ Bounds:      [-10.00, 35.00] → [5.00, 45.00]    │
   ├──────────────────────────────────────────────────┤
   │ Geometry Types:                                  │
   │   Point: 20 (48%)                                │
   │   Polygon: 22 (52%)                              │
   ├──────────────────────────────────────────────────┤
   │ Memory:      ~12 KB                              │
   ├──────────────────────────────────────────────────┤
   │ Properties (5 keys):                             │
   │   name: String                                   │
   │   population: Int                                │
   │   area: Double                                   │
   └──────────────────────────────────────────────────┘
   ```

8. **Property display limit** - show first 10 keys, then "... and N more" if exceeded.
