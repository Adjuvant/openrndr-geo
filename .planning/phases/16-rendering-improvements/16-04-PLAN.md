## Gap Analysis

### Gap 1: Geometry Normalization Not Integrated
**Root cause:** normalizePolygon() and normalizeMultiPolygon() exist but are never called.

**Impact:** Ocean.geojson data that crosses antimeridian renders with world-spanning artifacts.

**Fix locations:**
- `parsePolygon()` in GeoJSON.kt - must call normalizePolygon() and handle List<Polygon> return
- `parseMultiPolygon()` in GeoJSON.kt - must call normalizeMultiPolygon()

**Key challenge:** normalizePolygon() returns List<Polygon> when antimeridian splitting occurs, but parseGeometry() returns single Geometry. Solution: Flatten split polygons at the Feature level - when a polygon splits, create multiple features with the same properties.

### Gap 2: Holes Not Rendering in Standard Path
**Root cause:** DrawerGeoExtensions.kt ignores interior rings in standard rendering paths.

**Impact:** polygonsWithHole.geojson shows solid polygons without hole cutouts.

**Fix locations:**
- `Geometry.renderToDrawer()` for Polygon (lines 366-368) - only renders exterior
- `Geometry.renderToDrawer()` for MultiPolygon (lines 382-386) - only uses poly.exterior
- `projectGeometryToArray()` (lines 504-507) - only projects exterior coordinates
- `renderProjectedCoordinates()` (lines 526-548) - only renders exterior from cache

**Reference implementation:** OptimizedPolygon.renderOptimizedToDrawer() (lines 444-455) already correctly handles holes by checking interiors.isNotEmpty() and calling writePolygonWithHoles().

           // Collect all contours into a single Shape for proper hole rendering
           val allContours = mutableListOf<org.openrndr.shape.ShapeContour>()
           
           polygons.forEach { poly ->
               // Add exterior contour (clockwise for positive fill)
               val extPoints = poly.exterior.map { projection.project(it) }
               if (extPoints.size >= 3) {
                   val extContour = org.openrndr.shape.ShapeContour.fromPoints(extPoints, closed = true).clockwise

               val s = style ?: StyleDefaults.defaultPolygonStyle
               drawer.fill = s.fill
               drawer.stroke = s.stroke
               drawer.strokeWeight = s.strokeWeight
               drawer.shape(org.openrndr.shape.Shape(allContours))
