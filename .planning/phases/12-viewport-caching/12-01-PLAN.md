- Private var currentViewportState: ViewportState? = null

Primary method:
```kotlin
fun getProjectedCoordinates(
    geometry: Geometry,
    viewportState: ViewportState,
    projector: () -> Array<Vector2>
): Array<Vector2>
```

Implementation logic:
1. If viewportState != currentViewportState: cache.clear(), currentViewportState = viewportState
2. If geometry.isDirty (will be added in Plan 02): cache.remove(key), geometry.isDirty = false
3. If cache.size >= MAX_CACHE_ENTRIES: cache.clear()
4. val key = CacheKey(viewportState, geometry)
5. Return cache.getOrPut(key) { projector() }

Add clear() method to manually clear cache.
Add size property to expose cache size for testing.

Make class internal. Per 12-CONTEXT.md: Keep simple MutableMap (no ConcurrentHashMap needed - OpenRNDR rendering is single-threaded).
