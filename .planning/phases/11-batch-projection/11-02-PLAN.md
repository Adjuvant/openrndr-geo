   - `fun benchmarkProjection(geometry: Geometry, projection: GeoProjection, iterations: Int = 100): BenchmarkResult`

2. **Benchmark scenarios**:
   - Small LineString: 10 points
   - Medium LineString: 1,000 points
   - Large LineString: 10,000 points
   - Polygon: 5,000 points (complex ring)
   - MultiLineString: 3 lines of 1,000 points each

3. **Benchmark methodology** (from RESEARCH.md):
   ```kotlin
   // Warmup
   repeat(10) { perPointProjection(geometry, projection) }
   repeat(10) { batchProjection(geometry, projection) }
   
   // Per-point timing
   val perPointStart = System.nanoTime()
   repeat(iterations) { perPointProjection(geometry, projection) }
   val perPointDuration = (System.nanoTime() - perPointStart) / iterations
   
   // Batch timing
   val batchStart = System.nanoTime()
   repeat(iterations) { batchProjection(geometry, projection) }
   val batchDuration = (System.nanoTime() - batchStart) / iterations
   ```
