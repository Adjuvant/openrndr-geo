   - Function signature: `fun Sequence<Feature>.staggerByDistance(origin: Vector2, factor: Double = 10.0): Sequence<AnimationWrapper>`
   - Implementation: Calculate distance from origin, convert to delay = distance * factor
   - Use case: Ripple effects (farther features start later)
   - Return sequence of AnimationWrapper with computed delays

3. AnimationWrapper data class:

   - Spatial: `features.staggerByDistance(Vector2(0.0, 0.0), 10.0)` (10ms delay per unit distance)

- Create thousands of Animatable objects (global animator is better)
