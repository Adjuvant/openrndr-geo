## Decisions Made

1. **Wrap UnknownAuthorityCodeException in CRSTransformationException**
   - proj4j throws UnknownAuthorityCodeException for invalid EPSG codes
   - CRSTransformer catches and wraps in domain-specific CRSTransformationException
   - Provides consistent error handling across the projection API

2. **Case-insensitive CRS codes**
   - proj4j supports both "EPSG:27700" and "epsg:27700"
   - CRSTransformer uses `.lowercase()` for consistent behavior

3. **Helmert transformation accuracy expectations**
   - Tests use 0.05° tolerance (~5.5km) acknowledging Helmert's ~3-5m accuracy
   - Round-trip error verified to stay within 5 meters as expected
   - Creative coding use case doesn't require sub-meter OSTN15 accuracy
